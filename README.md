# BallsDex Merchant Package
This package is a custom package that allows buying balls in a shop. 
You can customize some features of the package in the `merchant/config.toml` file

> [!NOTE]
> This package is not compatible with BallsDex 3.X. And install first the package [Pack](https://github.com/Valen7440/Pack-Package/), otherwise this package will break!!!
